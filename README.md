# 💱 Currency Converter App

Ứng dụng Android chuyển đổi tiền tệ với tỷ giá thời gian thực, lưu lịch sử giao dịch và theo dõi xu hướng qua biểu đồ.

## 📱 Tính năng chính

- ✅ **Chuyển đổi tiền tệ** - Tỷ giá realtime từ ExchangeRate-API
- ✅ **Lịch sử giao dịch** - Lưu lại các lần chuyển đổi
- ✅ **Biểu đồ xu hướng** - Xem tỷ giá 7 ngày gần nhất
- ✅ **Cài đặt yêu thích** - Lưu các cặp tiền thường dùng
- ✅ **Offline support** - Dùng tỷ giá dự phòng khi mất mạng
- ✅ **Material Design** - Giao diện đẹp, thân thiện

## 🛠️ Công nghệ sử dụng

- **Ngôn ngữ:** Java
- **Database:** Room (SQLite)
- **Networking:** Retrofit, ExchangeRate-API
- **UI:** Material Design, CardView, RecyclerView
- **Chart:** MPAndroidChart
- **Architecture:** MVVM pattern, Clean Architecture

## 🚀 Cài đặt

1. Clone repository:
